КАК ЗАПУСТИТЬ TaskHub QA Sandbox
=====================================

ВАЖНЫЕ ПРАВИЛА ПЕРЕД ЗАПУСКОМ!
-------------------------------

❌ ЧТО ДЕЛАТЬ НЕЛЬЗЯ:
- Открывать HTML файлы напрямую в браузере (двойной клик)
- Вводить путь к файлу в адресной строке браузера
- Запускать приложение через file:// протокол

✅ ЧТО ДЕЛАТЬ НАДО:
- Всегда запускать сервер через START-SERVER.bat
- Открывать приложение через http://localhost:3000
- Если случайно открыли файл напрямую - следовать инструкциям

ВАРИАНТ 1: АВТОМАТИЧЕСКИЙ ЗАПУСК (САМЫЙ ПРОСТОЙ)
--------------------------------------------------

1. Найдите файл START-SERVER.bat в корне проекта
2. Дважды кликните по нему левой кнопкой мыши
3. Дождитесь сообщения "Starting server on http://localhost:3000"
4. Откройте браузер и перейдите на http://localhost:3000

ВАРИАНТ 2: РУЧНОЙ ЗАПУСК ЧЕРЕЗ КОМАНДНУЮ СТРОКУ
----------------------------------------------

1. Откройте командную строку (cmd.exe)
2. Перейдите в папку backend:
   cd "d:\для курсора\taskhub-qa-sandbox\backend"

3. Установите зависимости:
   npm install

4. Инициализируйте базу данных:
   npm run init-db

5. Запустите сервер:
   npm start

ВАРИАНТ 3: ЗАПУСК ЧЕРЕЗ ПОИСК
------------------------------

1. Нажмите Win + R
2. Введите: cmd
3. Нажмите Enter
4. Введите: cd /d "d:\для курсора\taskhub-qa-sandbox\backend" && npm start

ПРОВЕРКА РАБОТЫ
---------------

После запуска сервера откройте в браузере:
- http://localhost:3000 - Главная страница
- http://localhost:3000/login.html - Вход
- http://localhost:3000/register.html - Регистрация

Тестовые аккаунты:
- Логин: admin
- Пароль: admin123

ОСТАНОВКА СЕРВЕРА
-----------------

В окне командной строки нажмите Ctrl + C
Затем нажмите Y и Enter для подтверждения

УСТРАНЕНИЕ ПРОБЛЕМ
------------------

Если сервер не запускается:
1. Убедитесь, что установлен Node.js
2. Проверьте, что порт 3000 свободен
3. Попробуйте другой вариант запуска

Если возникают ошибки CORS или "file://":
- Это означает, что вы открыли файл напрямую
- Запустите сервер через START-SERVER.bat
- Откройте http://localhost:3000 в браузере

Если возникают ошибки:
- Закройте командную строку
- Удалите папку node_modules в backend
- Запустите npm install заново
